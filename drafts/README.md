# HAAM 약관 초안 — 배포 보류

> ⚠️ **이 폴더는 초안 전용입니다. 여기 있는 파일은 아직 배포되지 않았으며, 고객에게 효력이 없습니다.**

## 목적

HAAM 운영시스템이 현행 GAS+Google Sheets 기반에서 **Next.js 기반 자체 홈페이지(haam.co.kr)** 로 재구축됨에 따라,
이에 상응하는 **이용약관 v3** 및 **개인정보처리방침 v2** 초안을 사전에 준비합니다.

## 배포 조건

다음 3가지 조건이 **모두** 충족된 후에만 현행 문서를 교체 배포합니다.

1. **[P0] HAAM Next.js 홈페이지(haam.co.kr) 최종 구축 완료**
   - 회원가입·로그인·예약·결제·마이페이지·탈퇴 흐름 전부 프로덕션 동작 검증 완료
   - 관리자 대시보드의 회원/계약 관리 기능 확인 완료
2. **[P1] 실제 배포 프로세스에 초안 문구 반영**
   - 초안에 기재된 회원가입·탈퇴·휴면 절차가 코드로 구현되어 실제로 동작하는지 검증
   - 초안이 명시한 수탁업체(Supabase / Vercel / 토스페이먼츠 / 카카오) 와 실제 사용 스택이 일치하는지 확인
3. **[P2] 기존 회원 대상 사전 공지 및 경과 조치**
   - 시행일 **30일 전** 카카오 알림톡·SMS 개별 통지
   - 이용약관: 이의 미제기 시 동의 간주
   - 개인정보처리방침 중대 변경 부분: 시행일 이후 첫 로그인 시 재동의 화면 필수
